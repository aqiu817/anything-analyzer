# Anything Analyzer v3.5.0

## 新功能

- **MCP Tool: browser_screenshot** — 截取当前活动浏览器标签页的 PNG 截图，便于 AI Agent 进行视觉分析和调试
- **MCP Tool: filter_requests** — 支持按 HTTP 方法、域名、状态码（精确/范围）、Content-Type、URL 模式过滤已捕获的请求，避免返回全量数据
- **MCP Tool: cdp_send_command** — 暴露完整 Chrome DevTools Protocol 能力，支持调用任意 CDP 域（Page、DOM、Runtime、Network、Emulation、Input 等）
